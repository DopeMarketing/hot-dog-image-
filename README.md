# Hot dog image. 

## Overview
A simple mobile/web application that uses image recognition to determine whether a photo contains a hot dog or not. Users can take or upload photos and receive instant yes/no classification results.

## Problem
People need a quick and entertaining way to verify if food items in images are actually hot dogs

## Target User
General consumers who want a fun, novelty app for hot dog identification

## Features
- Image upload from camera or gallery
- Binary hot dog classification
- Simple yes/no result display
- Basic photo preprocessing
- Offline model capability
